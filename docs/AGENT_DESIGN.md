## 二、后端业务系统

### 2.1 Agent 请求处理流程

```
用户输入 → API Gateway → Intent Router → Agent Core → Tool Layer → Response
                                              │
                    ┌─────────────────────────┼──────────────────────┐
                    │  Mode | 触发条件          | 特点                │
                    │───────┼───────────────────┼─────────────────────│
                    │ rules | 无 API Key        | 快、免费、离线可用   │
                    │ llm   | 有 API Key        | 智能、需联网、有成本 │
                    │ auto  | 有 Key 用 LLM     | 自动降级到 rules    │
                    └─────────────────────────┴──────────────────────┘
```

### 2.2 API 设计

| 端点 | 方法 | 功能 | 认证 |
|------|------|------|------|
| `/api/agent/chat` | POST | 对话接口 | 无（公开） |
| `/api/agent/tools` | GET | 获取可用工具列表 | 无 |
| `/api/agent/history` | GET | 获取对话历史（按 sessionId） | Token |
| `/api/agent/config` | GET/PUT | 读写 Agent 配置 | Token |
| `/api/agent/stats` | GET | Agent 使用统计 | Token |

### 2.3 对话持久化方案

```typescript
// 存储结构
interface Conversation {
  sessionId: string
  visitorId: string      // 匿名访客 ID（fingerprint）
  startedAt: string
  lastActiveAt: string
  messages: Message[]
  metadata: {
    mode: 'rules' | 'llm'
    totalToolsUsed: number
    locale: 'zh' | 'en'
  }
}
```

**存储策略**：
- 开发环境：文件系统 JSON（`content/agent/conversations/`）
- Vercel 生产环境：客户端 localStorage + 服务端内存缓存
- 长期方案：Vercel KV / Upstash Redis

## 三、Agent 系统核心设计

### 3.1 核心架构

```
┌──────────────────────────────────────────────┐
│              Agent Core                       │

### 3.5 记忆机制

```
┌────────────────────────────────────────────┐
│            Memory System                    │
│                                             │
│  Short-term (当前会话)                      │
│  ├── 最近 N 轮对话（默认 10 轮）            │

│  └── 轮次耗尽 → 自动截断                    │

## 七、部署配置

### 环境变量

```bash
# .env.local（本地开发）或 Vercel Environment Variables

# Agent 模式：rules（免费）/ llm（智能）/ auto（自动）
AGENT_MODE=auto

# OpenAI 兼容 API
OPENAI_API_KEY=sk-xxxxxxxx
OPENAI_MODEL=gpt-4o-mini
OPENAI_BASE_URL=https://api.openai.com/v1

# 工具调用最大轮次
AGENT_MAX_TOOL_ROUNDS=3

# 站点信息
NEXT_PUBLIC_SITE_NAME=PM 思钱想厚
NEXT_PUBLIC_SITE_URL=https://www.aipmym.com
```

### 成本估算

| 模式 | 每次对话成本 | 月成本（1000次） |
|------|-------------|-----------------|
| rules | 免费 | ¥0 |
| gpt-4o-mini | ~¥0.02/次 | ~¥20 |
| gpt-4o | ~¥0.15/次 | ~¥150 |
| deepseek-chat | ~¥0.005/次 | ~¥5 |

> 推荐：日常用 auto 模式（有 Key 时用 LLM，Key 失效自动降级 rules），免费兜底有保障。
