# AI PM 个人作品网站 PRD

## 1. 产品概述

### 1.1 产品背景

为 AI 产品经理打造的专业个人作品展示网站，旨在通过现代化的交互体验和内容展示，提升个人品牌形象，吸引潜在客户、合作伙伴和雇主。

### 1.2 目标用户

- **潜在雇主**：招聘 AI 产品经理的企业 HR 和面试官
- **潜在客户**：需要 AI 产品咨询和外包服务的公司
- **行业同行**：产品经理社区的同行交流
- **合作方**：技术团队、设计团队等合作伙伴

### 1.3 核心价值主张

- **专业展示**：全方位展示 AI PM 的项目经验、专业能力和行业洞察
- **品牌塑造**：通过现代化设计和交互，塑造专业的个人品牌形象
- **高效沟通**：通过作品集和方法论文章，清晰传达专业能力
- **便捷联系**：提供多种联系方式，降低沟通门槛

## 2. 信息架构

### 2.1 页面结构

```
├── 首页
│   ├── Hero 区域（个人介绍 + CTA）
│   ├── 作品集模块（精选项目）
│   ├── 方法论模块（精选文章）
│   ├── 关于模块（个人信息 + 技能）
│   └── 联系模块（联系表单）
│
├── /portfolio（作品集列表页）
│   ├── 作品卡片网格
│   ├── 搜索功能
│   └── 分页功能
│
├── /portfolio/[slug]（作品详情页）
│   ├── 项目元信息
│   ├── Markdown 内容渲染
│   └── 外部链接跳转
│
├── /blog（方法论文章列表页）
│   ├── 文章卡片列表
│   ├── 搜索功能
│   └── 分页功能
│
└── /blog/[slug]（文章详情页）
    ├── 文章元信息
    ├── Markdown 内容渲染
    └── 分享功能
```

### 2.2 URL 设计规范

- `/` - 首页
- `/portfolio` - 作品集列表
- `/portfolio/[slug]` - 作品详情（slug 为项目英文缩写）
- `/blog` - 方法论列表
- `/blog/[slug]` - 文章详情（slug 为文章唯一标识）

## 3. 功能需求

### 3.1 首页模块

#### 3.1.1 Hero 区域

**功能描述**：展示个人核心信息和价值主张

**详细需求**：

| 需求项 | 描述 | 优先级 | 备注 |
|--------|------|--------|------|
| 个人标语 | 展示核心定位，如"热爱产品设计与技术探索" | P0 | 支持中英文 |
| 关键词高亮 | 自动识别并高亮关键词，显示对应表情符号 | P0 | 关键词池可配置 |
| CTA 按钮 | "查看更多项目" + "了解更多" | P0 | 滚动到对应区域 |
| 背景动画 | 粒子/渐变等动态效果 | P1 | 性能优先 |
| 响应式适配 | 移动端、平板、桌面端 | P0 | 媒体查询 |

**交互行为**：

- 关键词 hover 时显示表情符号 + 关键词的工具提示
- CTA 按钮点击后平滑滚动到对应区域
- 页面加载时逐段淡入动画

#### 3.1.2 作品集模块

**功能描述**：展示精选项目案例

**详细需求**：

| 需求项 | 描述 | 优先级 | 备注 |
|--------|------|--------|------|
| 项目卡片 | 展示缩略图、标题、简介 | P0 | 最多 3 个精选 |
| 项目标签 | 显示技术栈标签 | P0 | 可点击筛选 |
| 查看更多 | 跳转作品集列表页 | P0 | CTA 按钮 |
| 悬停效果 | 卡片悬停时放大 + 阴影 | P1 | 过渡动画 300ms |
| 点击跳转 | 点击卡片跳转到外部链接 | P0 | 新窗口打开 |

#### 3.1.3 方法论模块

**功能描述**：展示精选文章

**详细需求**：

| 需求项 | 描述 | 优先级 | 备注 |
|--------|------|--------|------|
| 文章卡片 | 展示标题、创建时间、简介 | P0 | 最多 3 篇精选 |
| 文章标签 | 显示分类标签 | P0 | 蓝色主题色 |
| 查看更多 | 跳转文章列表页 | P0 | CTA 按钮 |
| 悬停效果 | 卡片悬停时放大 + 阴影 | P1 | 过渡动画 300ms |
| 点击跳转 | 点击卡片跳转到文章详情页 | P0 | 站内跳转 |

#### 3.1.4 关于模块

**功能描述**：展示个人信息、技能和联系方式

**详细需求**：

| 需求项 | 描述 | 优先级 | 备注 |
|--------|------|--------|------|
| 个人信息 | 头像 + 核心信息 | P0 | 左对齐布局 |
| 统计数字 | 工作年限、项目数量等 | P0 | 大字号 + 动画 |

#### 3.1.5 联系模块（Footer）

**功能描述**：提供联系表单

**详细需求**：

| 需求项 | 描述 | 优先级 | 备注 |
|--------|------|--------|------|
| 联系表单 | 称呼 + 联系方式 + 备注 | P0 | 必填项校验 |
| 字数限制 | 称呼/联系方式 100 字，备注 1000 字 | P0 | 实时统计 |

| 消息推送 | 提交后发送到飞书群机器人 | P0 | Webhook 推送 |
| 推送内容 | 包含称呼、联系方式、备注、IP、设备、时间 | P0 | 纯文本格式 |

### 3.6 主题切换功能

**功能描述**：支持深色/浅色主题切换

**详细需求**：

| 需求项 | 描述 | 优先级 | 备注 |
|--------|------|--------|------|
| 主题检测 | 自动检测系统主题 | P0 | prefers-color-scheme |
| 主题切换 | 手动切换按钮 | P0 | 太阳/月亮图标 |
| 主题持久化 | 保存用户偏好到 localStorage | P0 | 优先读取 |
| 默认主题 | 无法识别系统主题时默认浅色 | P0 | light 主题 |
| 切换动画 | 平滑过渡效果 | P1 | 200ms 过渡 |

**主题配色方案**：

**浅色主题**：

- 背景色：`#ffffff`
- 文字色：`#000000` / `#333333` / `#666666`
- 主色调：`#f97316`（橙色）
- 次要色：`#3b82f6`（蓝色）
- 边框色：`#e5e7eb`

**深色主题**：

- 背景色：`#000000`
- 文字色：`#ffffff` / `#d1d5db` / `#9ca3af`
- 主色调：`#f97316`（橙色）
- 次要色：`#3b82f6`（蓝色）
- 边框色：`#1f2937`

## 5. 非功能需求

### 5.1 性能需求

| 指标 | 要求 | 说明 |
|------|------|------|
| 首屏加载时间 | < 3s | FCP < 3s |
| LCP | < 2.5s | Largest Contentful Paint |
| CLS | < 0.1 | Cumulative Layout Shift |
| FID | < 100ms | First Input Delay |
| TTI | < 3.5s | Time to Interactive |

**优化策略**：

- Next.js SSG/ISR 静态生成
- 图片懒加载
- 代码分割
- CSS/JS 压缩
- CDN 加速（Vercel Edge）

### 5.2 兼容性需求

| 浏览器 | 版本要求 |
|--------|---------|
| Chrome | >= 90 |
| Firefox | >= 88 |
| Safari | >= 14 |
| Edge | >= 90 |
| iOS Safari | >= 14 |
| Chrome Android | >= 90 |

### 5.3 响应式断点

| 设备 | 断点 | 说明 |
|------|------|------|
| 手机 | < 640px | sm: 及以下 |
| 平板 | 640px - 1024px | md: 及 lg: |
| 桌面 | > 1024px | xl: 及 2xl: |

### 5.4 可访问性需求

| 指标 | 要求 |
|------|------|
| 色彩对比度 | 文本与背景对比度 >= 4.5:1 |
| 键盘导航 | 所有功能可通过键盘操作 |
| ARIA 标签 | 按钮、链接等元素添加 aria-label |
| 焦点可见性 | 焦点状态清晰可见 |
