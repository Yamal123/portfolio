# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-05-15

### Added

#### 新增作品集列表页 (/portfolio)
- 创建完整的作品集子页面，支持所有项目展示
- 实现模糊搜索功能（支持标题、分类、标签、痛点搜索）
- 添加分页功能（每页6个项目）
- 包含返回首页导航
- 双语支持（中文/英文）
- 深色/浅色主题适配
- 新增3个项目案例：
  - 社交电商增长引擎
  - 智能数据分析平台
  - 智能风控系统

#### 新增方法论文章列表页 (/blog)
- 创建完整的文章子页面
- 实现模糊搜索功能（支持标题、标签、摘要搜索）
- 添加分页功能（每页6篇文章）
- 包含返回首页导航
- 双语支持（中文/英文）
- 深色/浅色主题适配
- 新增6篇方法论文章：
  - 产品数据驱动决策
  - 用户增长方法论
  - B端产品设计
  - AI时代产品经理能力模型
  - 产品经理如何做技术选型
  - 从0到1构建产品中台

#### 页面导航优化
- 作品集区域添加「查看更多项目」链接
- 博客区域添加「查看更多文章」链接
- 所有子页面添加返回首页导航

### Changed

#### 文案优化 (hero-section.tsx)
- 更新个人描述文案，强调产品设计和技术探索
- 新文案：「热爱产品设计与技术探索，专注供应链和AI Agent领域。热爱摄影、旅行与美食，奔赴山海，记录烟火。」
- 优化关键词映射，增加以下关键词：
  - 产品设计 🎨
  - 技术探索 🔬
  - 供应链 📦
  - AI Agent 🤖
  - 摄影 📷
  - 旅行 ✈️
  - 美食 🍜

#### 关键词系统优化
- 修复关键词重复渲染问题
- 优化 `renderDescriptionWithEmojis` 函数
- 实现按长度排序匹配（优先匹配长关键词）
- 添加重叠检测机制，避免重复高亮

### Fixed

#### Bug 修复
- 修复 `app/blog/page.tsx` 语法错误（缺少右括号）
- 优化组件导入（添加 Link 组件）
- 改进关键词匹配算法，避免重叠匹配
